# b-t-e-takip

Bu proje, Kişisel Finans Takip ve Çoklu Çalışma Alanı özelliklerini içerir.

## Özellikler
- **Çoklu Bütçe Yönetimi:** Farklı aylar veya projeler için izole alanlar (Workspaces).
- **Contextual Hotbar:** Bulunduğunuz ekrana göre (Dashboard, Yatırımlar, Kartlar vs.) değişen hızlı işlem paneli.
- **Güvenli Bütçe Sıfırlama:** Yanlışlıkla silinmeyi engelleyen onay pencereli (AlertDialog) veri temizleme mimarisi.

## Kurulum ve Çalıştırma

```sh
# Bağımlılıkları yükleyin
npm i

# Geliştirme sunucusunu başlatın
npm run dev
```

## Teknolojiler
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
